# FScruiserV2

FScruiser V2 (version 2) is the latest generation of timber cruising field data collection software for the U.S. Forest Serviceâ€™s National Cruise System. The mobile version of FScruiser is designed to operate on the Windows CE, Windows Mobile and Windows Embedded Handheld family of mobile operating systems.

FScruiser currently supports the cruise methods outlined in the Timber Cruising Handbook (FSH 2409.12), including fixed plot, fixed count-measure, point sampling, point count-measure, sample tree, classical 3P, fixed plot-3P, point-3P, 3P-point, and sample tree-3P.
